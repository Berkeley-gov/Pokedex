To search for a Pokemon you can enter a Pokemon's name (spelled correctly) or enter in a number ranging from 0 to 893.

For example, you could search for Bulbasaur by typing in his name in the search box below or enter Bulbasaur's index number 1. Either method will work.

The Pokedex will return information about a pokemon the user has searched for and it includes the pokemon's attack, defense, type, weight, and main move.

The Pokedex App uses the Poke API which grants the application access to an extensive database that details everything about the pokemon main game series. The Application uses the Fetch API and can only make get request for information and not manipulate it.
